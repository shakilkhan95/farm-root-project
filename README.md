# OrganicRoot - Farm-to-Table Organic Produce Landing Page

**OrganicRoot** (also referred to as FarmRoot) is a modern, responsive web application built with **React** and **Tailwind CSS**. It serves as a digital bridge between local farmers and health-conscious consumers, offering 100% organic, handpicked produce delivered directly to the doorstep.

## 🚀 Key Features

- **Dynamic Product Filtering:** A categorized catalog allowing users to browse items across Food & Drinks, Fresh Fruits, Vegetables, and Dried Foods & Nuts.
- **Component-Based Architecture:** Highly modular design with reusable components like `ProductCard`, `ButtonPrimary`, and dedicated sections for a clean codebase.
- **Interactive UI/UX:**
    - **Testimonial Slider:** An autoplaying customer review section built for high engagement.
    - **Process Transparency:** A "How It Works" section detailing the journey from farm harvest to eco-friendly packaging.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using Tailwind's utility-first framework.

## 🛠️ Technical Stack

- **Frontend:** [React.js](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) (FaLeaf, FaShippingFast, etc.)
- **Fonts:** Poppins (Google Fonts)

## 📂 Project Structure

The project follows a clean directory structure for maintainability:

```text
src/
├── assets/img/        # Product, hero, and testimonial images
├── components/        # Specialized UI components
│   ├── About/         # Mission and brand history
│   ├── Features/      # Core value propositions
│   ├── Hero/          # Landing section with CTA
│   ├── HowItWorks/    # Operational workflow
│   ├── Products/      # Product grid and filtering logic
│   └── Testimonial/   # Customer reviews slider
├── data/              # Centralized product and feature data
└── App.jsx            # Main application entry point
```

## 📋 Core Value Propositions

Based on the application's "Features" module, the platform emphasizes:
1.  **100% Organic:** Grown without harmful chemicals or pesticides.
2.  **Fast Delivery:** Doorstep delivery within hours of harvesting.
3.  **Farmer Support:** Fair trade and sustainable practices for local communities.
4.  **Quality Assured:** Rigorous checks for freshness before every delivery.

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shakilkhan95/organic-root.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
